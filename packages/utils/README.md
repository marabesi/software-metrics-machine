# @smm/utils

Shared utility functions for Software Metrics Machine.

## Development

```bash
# Install dependencies
pnpm install

# Build
pnpm build

# Run tests
pnpm test

# Generate coverage report
pnpm test:coverage

# Lint code
pnpm lint

# Type check
pnpm type-check
```

## Architecture

Contains shared utility functions used across:
- @smm/core
- @smm/rest
- @smm/cli

Examples:
- Date and time utilities
- String manipulation
- Data transformation helpers
