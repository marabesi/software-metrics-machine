# @smm/core

Core business logic package for Software Metrics Machine.

## Development

```bash
# Install dependencies
pnpm install

# Build
pnpm build

# Run tests
pnpm test

# Generate coverage report
pnpm test:coverage

# Lint code
pnpm lint

# Type check
pnpm type-check
```

## Architecture

Contains the core business logic for:
- Git repository analysis
- Code metrics calculation
- Deployment frequency tracking
- Pull request analytics

## Migration Notes

This package will contain the TypeScript migration of the Python `api/src/software_metrics_machine/` module.
